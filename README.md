To deploy this project, follow these steps:

1. **Backend**:
   - Install dependencies:
     ```bash
     npm install
     ```
   - Navigate to the backend directory:
     ```bash
     cd backend
     ```
   - Start the backend:
     ```bash
     npm start
     ```

   - Install dependencies:
     ```bash
     npm install
     ```
   - Navigate to the frontend directory:
     ```bash
     cd frontend
     ```

     ```bash
     npm run dev
     ```
